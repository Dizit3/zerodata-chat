# Инструкции для ИИ-ассистентов

В этой папке содержатся ключевые документы, описывающие текущее состояние проекта, архитектурные решения и историю изменений после масштабного рефакторинга.

## Состав документов:
- **[task.md](task.md)**: Список выполненных и текущих задач.
- **[implementation_plan.md](implementation_plan.md)**: План рефакторинга UI и архитектуры.
- **[walkthrough.md](walkthrough.md)**: Полный отчет о внесенных изменениях, исправленных багах и верификации.

## Важные архитектурные решения:
1. **SOLID**: Проект следует принципам SRP и ISP. Интерфейс `MqttManager` разделен. Сериализация вынесена в `MqttPayloadMapper`.
2. **UI**: Используется Jetpack Compose с глубокой декомпозицией на компоненты (`ui/components`).
3. **DI**: Все зависимости управляются через Koin (`di/AppModule.kt`). Явное указание типов в `get()` критично для стабильности.
4. **Persistence**: Используется Room. Репозитории работают с `StateFlow`.

Используйте эти файлы как основной источник правды (Source of Truth) перед началом новых задач.
